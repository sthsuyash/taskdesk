# Getting Started

## Installation

```bash
# Install frontend dependencies
cd ui && pnpm install

# Install server dependencies
cd server && pnpm install
```

## Development

```bash
# Start server (port 8000) - in one terminal
cd server && pnpm dev

# Start frontend (port 5173) - in another terminal
cd ui && pnpm dev
```

## Production Build

```bash
# Build both frontend and server
pnpm build

# Start production server
pnpm start
```

The production build serves the frontend at `/app` and the API at `http://localhost:8000`.
