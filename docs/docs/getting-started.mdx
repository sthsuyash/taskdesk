# Getting Started

## Installation

```bash
pnpm install
```

## Development

```bash
# Start both frontend and server
pnpm dev

# Or run individually
pnpm dev:ui      # Frontend: http://localhost:5173
pnpm dev:server  # Server: http://localhost:8000
```

## Production Build

```bash
pnpm build
pnpm start
```

The production build serves the frontend at `/app` and the API at `http://localhost:8000`.
