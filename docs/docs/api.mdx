# API Reference

## REST Endpoints

### Create Session

```http
POST /api/sessions
```

Response:
```json
{ "sessionId": "uuid" }
```

### Append Events

```http
POST /api/sessions/:id/events
```

Body:
```json
{ "events": [...] }
```

### Get Session Events

```http
GET /api/sessions/:id/events
```

### List Sessions

```http
GET /api/sessions
```

### Delete Session

```http
DELETE /api/sessions/:id
```

### List Tasks

```http
GET /api/tasks
```

Response:
```json
{ "tasks": [{ "id": "uuid", "title": "string", "description": "string", "status": "todo|done", "createdAt": timestamp, "updatedAt": timestamp }] }
```

### Create Task

```http
POST /api/tasks
```

Body:
```json
{ "title": "string", "description": "string", "status": "todo|done" }
```

Response (201):
```json
{ "task": { "id": "uuid", "title": "string", "description": "string", "status": "todo|done", ... } }
```

### Update Task

```http
PUT /api/tasks/:id
```

Body:
```json
{ "title": "string", "description": "string", "status": "todo|done" }
```

### Delete Task

```http
DELETE /api/tasks/:id
```

## WebSocket

### Connect

```
ws://localhost:8000/live?session=<id>&role=viewer
ws://localhost:8000/live?session=<id>&role=recorder
```

### Role: Recorder

Send events:
```json
{ "type": "event", "data": {...} }
```

### Role: Viewer

Receive events in real-time.
